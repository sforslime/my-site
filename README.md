# my-site

Personal portfolio site for Ayo — built with Next.js 15, Tailwind CSS, and deployed on Vercel.

## Pages

- `/` — Home: intro, roles, skills, animated sphere
- `/about` — About
- `/work` — Work / projects
- `/contact` — Contact

## Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Custom Typewriter, WireframeSphere, ScrollFadeText components
- **Deployment:** Vercel (auto-deploys on push to `main`)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
